Explanation of AJAX

This page demonstrates an example of AJAX is used when the Get CD Collection button is clicked. A table of song information get displayed through the following:

  • Creating XMLHttpRequest object to load cd_catalog.xml.
  • Defining a Callback function, myFunction(xml), to handle the response with modifying the Artist name to developer name, and year to current year, and retrieving title through getElementsByTagName
  • Sending modified response data in XMLHttpRequest object back to the web server with update without reloading the page.

Click on the Get CD Collection to load data from the XML file.


REST API

Imgflip API returns a list of image memes or provide user ability to add text or caption to mems directly.

  • Intergate mems into Slack
  • Automatically caption one of the top 2,048 memes with search_memes api
